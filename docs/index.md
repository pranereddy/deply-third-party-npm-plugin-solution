# Depsight Third-Party Plugin

The *Depsight Thrird Party Plugin* project is part of [modern-python-devops-engineering](https://github.com/ValentinTwin1206/modern-python-devops-egineering) 
course and should be used in this context.

This site contains the hands-on instructions for building, packaging, and
publishing a third-party plugin for the
[Depsight](https://pypi.org/project/depsight/) dependency analysis framework.
